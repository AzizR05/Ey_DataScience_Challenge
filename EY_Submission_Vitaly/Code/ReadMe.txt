How To Run:
This is a Python3 Jupter Notebook
Once these are installed on your machine 
go to this folder in your terminal and run Jupyter Notebook.
This should automatically open a homepage in your browser 
where you can select the submission file.

The first part was just to look through the code,
if the code needs to be run then the competition datasets need
to be downloaded and placed in this folder.
The imports stated in the first cell of the notebook must also
be installed for the program to work.

2 Models are added 
Higher Scoring Model -> Classifier_Submission.bin
Uses XGBoost Classification

The other two models are Regression Models 
X_Regressor_1.bin is a Longitude Regressor
Y_Regressor_1.bin is a latititude Regressor
